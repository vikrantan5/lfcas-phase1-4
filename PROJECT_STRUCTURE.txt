Database Collections:
====================
1. users - User authentication and profiles
2. advocates - Professional advocate profiles
3. cases - Legal case records
4. hearings - Court hearing schedules
5. documents - Document metadata
6. messages - Real-time case communication
7. notifications - User notifications
8. ratings - Advocate ratings and reviews
9. admin_logs - Platform manager activity
10. groq_ai_logs - AI query logs

Tech Stack:
===========
Backend:
- FastAPI (Python 3.11)
- MongoDB Atlas (Motor async driver)
- Groq API (LLaMA 3.3 70B)
- Cloudinary (File storage)
- Socket.IO (Real-time messaging)
- JWT (Authentication)
- bcrypt (Password hashing)

Frontend (Phase 5):

- Tailwind CSS

- React Router v7
- Axios
- Socket.IO Client
